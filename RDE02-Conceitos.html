1. Quais foram as principais invenções da década de 1990?
  • O cientista Tim Berners Lee criou em 1992 a World Wide Web.
  • Surgimento de navegadores como o Internet Explorer, Netscape, Mozilla Firefox, Google Chrome, Opera e Lynx.
  • URL 1994.
  • Javascript 1995.
  • Google 1998.

2. Explique os protocolos http e https.
HTTP significa Protocolo de Transferência de Hipertexto é um protocolo de comunicação utilizado para sistemas de informação de hipermídia, distribuídos e colaborativos. Ele é a base para a comunicação de dados da www. Já o HTTPS possui uma camada a mais de segurança utilizando o protocolo SSL/TLS de forma em que os dados são transmitidos por meio de uma conexão criptografada e é verificada a autenticidade do servidor e do cliente.

3. Qual a função do DNS e por que a necessidade de sua existência?
DNS significa Sistema de Nomes de Domínio, serve para associar nomes de domínio mais facilmente memorizáveis a endereços de IP numéricos. Por exemplo, se eu digitar www.google.com entrará no Google, se eu digitar 172.217.28.228 também entrará no Google, mas pelo nome é mais fácil de lembrar. O servidor DNS recebe o nome do domínio e entrega o local onde está determinada página que está pesquisando.

4. Caso não existisse o protocolo TCP/IP quais problemas aconteceriam?
Sem este protocolo não teria uma confiabilidade nas troca de informações, já que ele garante que as informações cheguem completas de um local para outro.

5. Defina HTML
Significa Linguagem de Marcação de Hipertexto. O HTML pode ser interpretado pelos navegadores e serve para a criação de páginas na Web.

6. Acesse o site do projeto W3 e descreva qual era o propósito do projeto.
Este é o primeiro site que surgiu na www e descreve detalhes da World Wide Web, ideia do próprio Berners-Lee. A WWW tinha como objetivo inicial permitir que os profissionais do CERN trocassem informações científicas a partir de seus próprios computadores. O site do The Project serviu de teste para isso.
