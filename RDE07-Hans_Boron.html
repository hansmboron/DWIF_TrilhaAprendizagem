O que são Branches?

São ramificações de um projeto de software ou documento e tem como objetivo o desenvolvimento paralelo de:

  • Novas funcionalidades
  • Correção de bugs
  • Alteração de código

O nome da branck principal quando um repositório é criado se chama main ou master.

Não é recomendado trabalhar diretamente na branch main apesar de ser possível fazer isso. O que se recomenda é criar uma branch segundaria para fazer todas as alterações necessárias e depois fazer um merge para a branch main quando estiver estável.

As branches do git pussibilitam com que vários times de desenvolvimento possam trabalhar em um mesmo projeto ao mesmo tempo, cada um em sua branch podendo depois fazer o merge sem grandes problemas, o que seria muito complicado de fazer seu usar um sistema de versionamento.


Como hospedar meu site simples gratuitamente?

GitHub Pages:

É possível hospedar seu site desde que ele não tenha banco de dados e esteja como público no GitHub gratuitamente e de forma bem simples. Na página do seu repositório basta clicar em configurações e encontrar Github Pages para ativar.

Depois de ativado será gerada uma URL para você acessar seu site parecida com esta:
https://nomeusuario.github.io/repositorio
