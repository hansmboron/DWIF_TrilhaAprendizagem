CSS (Cascading Style Sheets)

Para que serve?

Definição CSS:

Cascading Style Sheets - Folhas de Estilo em Cascata - Tem por objetivo realizar a Estilização de conteúdos dde documento HTML. Podem definir fontes, cores, espaçamento dentro outros.

Exemplo:


seletor {
    propriedades: valor;     /*comentário*/
    propriedades-subpropriedades: valor;
}
/*
seletor - seleciona elemento HTML com base no nome do elemento ao qual o estilo será aplicado
propriedades - o aspecto específico do elemento que você quer usar como estilo
valor - a configuração aplicada ao atributo
*/

/*personalizando tags p*/
p {
    color: purple;
    font-weight: bold;
    text-align: center;
}

Três formas de utilizar o CSS

Inline

  • Os códigos CSS são definidos dentro de uma tag HTML.
  • Afetam somente a tag em que foi definida.

<h3 style="font-size: larger; margin-top: 2em; color: darkgreen;">Exemplo inline</h3>

Incorporado

  • Os códigos CSS são definidos no cabeçalho dentro da tag HEAD.
  • Afetam somente o ducumento em que foi definido.

<head>
    ...
    <style>
        body {
            font-family: "Noto Sans", serif;
        }
        h1, h2 {
            color: darkred;
        }
    </style>
</head>

Externo

  • Os códigos CSS são definidos em um arquivo externo.
  • Afetam todos os documentos que estão linkados ao CSS.

/* style.css */

body {
    font-family: "Noto Sans", serif;
}
h1, h2 {
    color: darkred;
}
                

<!-- index.html -->

...
<head>
    <link href="style.css" rel="stylesheet">
</head>
...
