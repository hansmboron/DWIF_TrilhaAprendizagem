Softwares de Versionamento

O que é Git?

O Git é um sistema open-source de controle de versão utilizado pela grande maioria dos desenvolvedores atualmente. Com ele podemos criar todo histórico de alterações no código do nosso projeto e facilmente voltar para qualquer ponto para saber como o código estava naquela data. O git foi criado por Linus Torwalds o famoso criador do Linux.

Alguns dos principais comandos do Git

  • commit - Torna permanente um conjunto de alterações em um repositório.
  • push - Envia alterações para o servidor no repositório remoto.
  • fetch - Verifica se seu repositório local esta em sincronia com o remoto.
  • pull - Sincroniza o repositório local com o remoto atualizando as versões.

O que é GitHub?

O Github é um repositório remoto, criado como um serviço online de hospedagem de repositórios do Git.
Link: GitHub

Vantagens em utilizar o Git e o GitHub

  • Trabalho em equipe - Pode trabalhar com várias pessoas ao mesmo tempo em um mesmo repositório sem problemas.
  • Segurança - Se estragar seu computador você não perde as alterações que fez no repositório já que uma cópia está no servidor remoto do GitHub.
  • Gestão de mudanças - Se você fizer alterações desastrosas no seu código, poderá 'voltar no tempo' para algum commit anterior onde seu projeto estava funcionando corretamente.
