Exemplificando algumas Tags HTML

<table>

A tag <table> define uma tabela em HTML.

Cada linha da tabela é definida com a tag <tr>. Cada cabeçalho da tabela é definido com a tag<th>. Cada célula da tabela é definida com a tag <td>

Exemplo:

    <table>
      <tr>
        <th>Nome</th>
        <th>Sobrenome</th>
        <th>Idade</th>
      </tr>
      <tr>
        <td>Jill</td>
        <td>Smith</td>
        <td>50</td>
      </tr>
      <tr>
        <td>Eve</td>
        <td>Jackson</td>
        <td>94</td>
      </tr>
    </table>
Nome Sobrenome Idade
Jill Smith 50
Eve Jackson 94

<del>

A tag <del> define que algum texto foi deletado do documento. Normalmente os navegadores colocam um risco no meio da palavra ou frase

Exemplo:

    <p>Minha cor favorita é <del>azul</del> <ins>verde</ins>!</p>

Minha cor favorita é azul verde !


<ins>

A tag <ins> define que algum texto foi inserido no documento. Normalmente os navegadores representam isso colocando u underline abaixo do texto

Exemplo:

    <p>Minha cor favorita é <del>azul</del> <ins>verde</ins>!</p>

Minha cor favorita é azul verde !


<sub>

A tag <sub> define um subtexto. Que é um texto menor que aparece abaixo do nível normal dos textos comuns.

Exemplo:

    <p>Este texto contém um <sub>subtexto</sub>.</p>

Este texto contém um subtexto.


<sup>

A tag <sup> define o texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé.

Exemplo:

    <p>Este texto contém um texto <sup>sobretexto</sup>.</p>

Este texto contém um texto sobretexto.


<blockquote>

A tag <blockquote> especifica uma sessão de é citada de outra fonte.

Exemplo:

      <blockquote cite="http://www.worldwildlife.org/who/index.html">
        For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization...
      </blockquote>
For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

<q>

A tag <q> define uma citação curta. Normalmente o Navegador coloca "aspas" por volta da citação.

Exemplo:

        <p>
        O Objetivo da WWF é:
            <q>Construir um futuro onde as pessoas vivam em harmonia com a natureza.</q>
            Nós esperamos que ela tenha sucesso.
        </p>

O Objetivo da WWF é: Construir um futuro onde as pessoas vivam em harmonia com a natureza. Nós esperamos que ela tenha sucesso.


<abbr>

A tag <abbr> define uma abreviação ou um acrónimo, por exemplo: HTML, CSS, OMS, etc.

Exemplo:

        A <abbr title="Organização Mundial da Saúde">OMS</abbr> foi fundada em 1948.
A OMS foi fundada em 1948.

<address>

A tag <address> define o contato de informação do autor de um documento ou artigo.

Exemplo:

        <address>
            Written by <a href="mailto:webmaster@example.com">Hans M. Boron</a>
        </address>
Written by Hans M. Boron

<cite>

A tag <cite> define um título de alguma obra. Exemplo: Uma música, um poema, um livro, uma pintura, etc.

Exemplo:

        <address>
             <p><cite>O Grito</cite> by Edward Munch. Pintado em 1893.</p>
        </address>

O Grito by Edward Munch. Pintado em 1893.


<bdo>

BDO significa Bi-Directional Override. Serve para mudar a direção do texto

Exemplo:

        <bdo dir="rtl">
            Este texto vai ir da direita para a esquerda.
        </bdo>
Este texto vai ir da direita para a esquerda.

<code>

A tag <code> define um pedaço de código de computador. O navegador usa a fonte monospace padrão.

Exemplo:

        <code>
            print(Hello World!)
        </code>
print(Hello World!)

<pre>

A tag <pre> define texto pré-formatado. Preserva os espaços e quebras de linha.

Exemplo:

        <pre>
            Preserva os    espaços
               e quebras
                    de linha
        </pre>
        
            Preserva os    espaços
               e quebras
                    de linha

<kbd>

A tag <kbd> define inputs de teclado. Normalmente mostrado com fonte monospace.

Exemplo:

        <p>Precione <kbd>Ctrl</kbd> + <kbd>C</kbd> para copiar texto.</p>

Precione Ctrl + C para copiar texto.
