Hyperlinks

Definição

A tag HTML < a > serve para definir links que redirecionam para outras páginas ou para conteúdos da mesma página. Para isso basta emvolver a palavra ou frase que quer transformar em link com a tag < a > e colocar uma localização no atributo href.

O atributo mais importante da tag < a > é a href pela qual é possível indicar a localização do link aonde voçê quer chegar.

Abrir link em página diferente

Se quiser que o link abra em uma nova guia, em vez de abrir na mesma como aconteçe por padrão pode escrever _blank no atributo target da tag < a >.
