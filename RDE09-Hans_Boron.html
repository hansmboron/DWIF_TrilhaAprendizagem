Multimídias

Tag iframe

A tag <iframe> é usada para incorporar outro documento ao documento HTML.

Exemplo:

Incorporar um site dentro do site:


<iframe src="https://irati.ifpr.edu.br" height="300" width="100%"></iframe>

Tag video

A tag <video> serve para colocarmos um vídeo no HTML5.

Exemplo:


<video poster="video/Bike.jpg" width="320" height="240" autoplay muted>
    <source src="video/_Bike.mp4" type="video/mp4">
    <source src="video/Bike.webm" type="video/webm">
    <source src="video/Bike.ogv" type="video/ogv">
    Seu navegador não suporta este vídeo!
</video>

Tag audio

A tag <audio> serve para colocarmos um áudio no HTML5.

Exemplo:


<audio preload="metadata" controls>
    <source src="audio/Hey_Sailor_-_Letter_Box.ogg" type="audio/ogg">
    <source src="audio/Hey_Sailor_-_Letter_Box.mp3" type="audio/mpeg">
    <source src="audio/Hey_Sailor_-_Letter_Box.wav" type="audio/wav">
    Seu navegador não suporta este áudio!
</audio>

Tag picture

A tag <picture> serve para agrupar imagens. Normalmente utilizada para direcionar resposividade no site.

Exemplo:


<picture>
    <source media="(max-width:400px)" srcset="img/bike_110.jpg">
    <source media="(max-width:700px)" srcset="img/bike_320.jpg">
    <source media="(max-width:1000px)" srcset="img/bike_640.jpg">
    <img src="img/bike_full.jpg" alt="Mtb Por-do-Sol">
</picture>
